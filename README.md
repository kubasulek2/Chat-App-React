# Chat-App-React

### Cool Chat App made with react, and node.js

#### Features:
* Create custom rooms, join to existing ones.
* Enjoy public or private conversations.
* Receive notification on new messages
* Block annoying users.
* Enjoy cool looking UI

#### Technologies:
* react
* node.js
* socket.io
* material-UI
* express
* immer
* emojiMart
